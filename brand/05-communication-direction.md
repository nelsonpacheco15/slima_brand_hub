# Communication direction — illustration only

**Going forward, Slima communicates through one visual language: the editorial line-illustration post.**

Hand-drawn black-ink line illustrations (New Yorker / editorial-cartoon feel), on white, with an IBM Plex Serif headline, a single indigo accent, and the ✳ Slima mark in the footer. Witty, literary, understated. See the full spec in `01-brand-guidelines.md` → *Editorial line-illustration + wordplay*.

**Retired — do not reuse:** the old mixed style we used to communicate with (AI-generated hero images from ChatGPT/Gemini, raw product screenshots as posts, one-off PSDs). Those have been removed from this source of truth. Only illustration posts represent Slima now.

## Where the illustration work lives
- `../social-media/ads/` — the line-illustration ad system (`ads.html` builder + exported PNGs in `ad-images/`). This is the reference execution of the style.
- `../social-media/posts/` — illustration campaign posts, kept as editable Illustrator/Affinity source (`.ai` / `.af`) plus the exported carousels (`1x/`). Campaigns: Authors, review, script studio, version branching, announcement (Discord), get feedback, TEDx.

## Rule of thumb
If a post isn't a line illustration in the Slima style, it's off-brand. Build new posts from the `ads.html` system or the campaign `.ai` files, not from screenshots or generated imagery.
